Tentang Aplikasi

Tutor Aljabar Linear adalah aplikasi web interaktif untuk belajar aljabar linear langkah demi langkah.

Fitur:

  • Penjelasan konsep dengan notasi matematika
  • Contoh interaktif dengan perhitungan langkah demi langkah
  • Kalkulator untuk latihan
  • Kuis untuk menguji pemahaman

Topik yang Tersedia:

  • 📐 Vektor - Konsep dasar, operasi, dot product
  • 📊 Matriks - Operasi, determinan, invers
  • 📝 Sistem Persamaan - Substitusi, eliminasi, Cramer
  • 🔢 Nilai Eigen - Eigenvalue dan eigenvector

Teknologi:

  • Backend: Flask (Python)
  • Frontend: HTML, CSS, JavaScript
  • Matematika: NumPy, MathJax
